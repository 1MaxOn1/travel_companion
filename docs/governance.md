## 2. Политика логов и работа с персональными данными

**Персональные данные**  
В PoC система не собирает и не обрабатывает персональные данные пользователей, за исключением предпочтений, которые сохраняются в Memory Module. Эти предпочтения обезличены: они привязаны к анонимному идентификатору сессии, не требующему регистрации. В будущих версиях может потребоваться согласие на обработку.

**Политика логов**  
- Все логи сохраняются локально в рамках демонстрационной среды, не передаются третьим сторонам.  
- Идентификаторы сессий не содержат личной информации.  
- Логируются только обезличенные действия агентов (вызовы инструментов, извлечённые структурированные данные) для отладки и оценки метрик.  
- Запросы пользователя логируются в обезличенном виде без привязки к реальному лицу.

## 3. Защита от инъекций и подтверждение действий

**Защита от Prompt Injection**  
Router-агент (Orchestrator) извлекает из сообщения пользователя структурированные поля:  
- даты (start_date, end_date)  
- бюджет (budget_amount, budget_currency)  
- интересы (interests)  
- ограничения (dietary, visa_free и т.д.)  

Все эти поля валидируются по типу и формату перед передачей downstream-агентам. Свободный текст пользователя никогда не конкатенируется в системные промпты агентов напрямую. Для анализа контекста используется только извлечённый JSON.

**Подтверждение критических действий**  
Система не осуществляет реальное бронирование, поэтому подтверждения оплаты не требуются. Однако при изменении ключевых параметров (смена направления, дат, существенное изменение бюджета) агент запрашивает явное подтверждение пользователя перед перестроением маршрута. Например:  
*«Я предлагаю заменить Италию на Турцию, чтобы уложиться в бюджет. Согласны?»*

**Disclaimer**  
В начале каждой сессии и в итоговом отчёте выводится дисклеймер:  
*«Система находится в режиме демонстрации (PoC). Все данные о билетах, отелях и событиях являются тестовыми и не гарантируют актуальности. Система не осуществляет реальное бронирование и не несёт ответственности за решения, принятые на основе предоставленной информации.»*
