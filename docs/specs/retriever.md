# Спецификация поискового модуля (retriever)

## Назначение
Поиск актуальной туристической информации: авиабилеты, отели, достопримечательности, рестораны, погода, визовые требования.

## Реализация
- Файл: `agent/tools.py`
- Функция: `search_internet(query: str) -> dict`

## Параметры запроса
| Параметр | Тип    | Обязательный | Описание                     |
|----------|--------|--------------|------------------------------|
| query    | string | да           | Поисковый запрос (не пустой) |

## Возвращаемый формат
Успешный ответ:
```json
{
  "source": "DuckDuckGo",
  "results": [
    {"title": "Заголовок", "link": "https://...", "snippet": "Описание"}
  ]
}
